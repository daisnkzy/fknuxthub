## Setup

Make sure to install the dependencies:

```bash
# pnpm (recommend)
pnpm install

# npm
npm install

# yarn
yarn install

# bun
bun install
```

## Generate the Migrations
```bash
npm run db:generate
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# pnpm (recommend)
pnpm dev

# npm
npm run dev

# yarn
yarn dev

# bun
bun run dev
```

## Production

Build the application for production:

```bash
# pnpm (recommend)
pnpm run build

# npm
npm run build

# yarn
yarn build

# bun
bun run build
```

Locally preview production build:

```bash
# pnpm (recommend)
pnpm run preview

# npm
npm run preview

# yarn
yarn preview

# bun
bun run preview
```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
